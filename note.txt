
Mes commandes commandes git à ne pas oublier

    Liste de toutes les branches
        git branch -a
    les différences entre la branche de nom une-branche et la branche main
        git diff une-branche main
    Basculer vers la branche de nom une-branche
        git switch une-branche
        git checkout une-branche

Voir les différences entre la branche de nom une-branche et la branche main
git diff une-branche main
Créer une branche de nom une-branche
git branch une-branche

Créer une branche de nom une-branche et basculer vers cette branche
git switch -c une-branche
git checkout -b une-branche

    Renommer une branche 
         git branch -m une-branche autre-branche
    Supprimer une branche de nom une-branche
        git branch -d une-branche
    Fusionner la branche main avec la branche de nom une-branche
    git checkout main
    git merge une-branche -m "Explication"
